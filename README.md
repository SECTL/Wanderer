## 功能

### 快速管理考勤

- [x] 支持通过拼音搜索、排序人员列表，快速找到要修改状态的人员
- [x] 支持查看、修改历史的考勤记录

### 考勤状态查看

- [ ] 即将支持查看指定学生历史状态，进行汇总
