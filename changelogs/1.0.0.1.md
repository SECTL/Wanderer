这是 WandererAttendance (易考勤) 的第一个正式版。

> 1.0.0.0 用于测试持续集成。

## 💫 亮点

毫无亮点。支持 Windows + Linux (未测试) + macOS (未测试) + Android + Browser 五个平台。~~全靠 Avalonia 的兼容性~~

使用了 FluentAvalonia + CommunityToolkit.Mvvm + Microsoft.Extensions.Hosting 作为地基。

名单支持通过 Excel(暂不支持多个工作表...)、CSV、txt 导入。

## 🚀 新增功能

- 【核心】添加基本界面框架
- 【核心】添加基本模型
- 【字体】使用 Harmony OS Sans Sc
- 【平台】兼容 Browser 平台
- 【界面】添加「主页」「考勤」「档案」「历史记录」「关于」界面
- 【设置】添加「考勤状态修改样式」设置
- 【集成】支持 GitHub Workflow 自动构建、
- 【版本】使用 ClassIsland.SimpleGitInfoGenerator 追踪版本

# 🐛 问题修复

- 【平台】解决了安卓上不显示状态栏的问题
